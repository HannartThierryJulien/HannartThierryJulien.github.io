Langages

IDE

NetBeans

Réalisé

En 2ème année de bachelier pour le cours de "Programmation orientée objet" (mai-juin 2022).

Description

À partir des données contenues dans deux fichiers XML, ce programme permet de prendre la commande d'une table de restaurant, de lister les commandes stockées dans la base de données et de créer divers graphiques.
L'interface utilisateur a été conçue manuellement avec la bibliothèque graphique Swing.
