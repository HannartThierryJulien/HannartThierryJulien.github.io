Langages

Java, SQL, HTML, CSS, Javascript

IDE

Eclipse

Visual Studio Code pour la partie web

Réalisé

En 3ème année dans le cadre du cours de "Projet de développement SGBD" (septembre-février 2023).

Description

Ce programme a pour but de permettre la gestion d'articles et de commandes depuis une interface affichée sur un navigateur.
Il a été élaboré à l'aide des framework Spring Boot et Hibernate. Le moteur de template Thymeleaf a également été utilisé.
Les données sont stockées dans une base de données MySql.
