Description

Ce programme est entrain d'être réalisé pour le cours de "Projet de développement SGBD".
Il a pour but de faire la gestion d'articles et de commandes depuis une interface affichée sur un navigateur.
Spring Boot est utilisé.
Les données sont stockées dans une base de données MySql.
